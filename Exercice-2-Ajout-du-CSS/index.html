Les bases de l'HTMLsource

Les balises

Les balises paires

Les balises paires sont des balises qui fonctionnent par paire, forcément !
La première balise est ce qu'on appelle une balise "ouvrante", et la seconde balise est la balise "fermante".

Les balises paires fonctionnent comme ceci :

En premier lieu, on écrit la balise qui sera constitué des caractères suivant: < >
On écrit à l'intérieur le nom de la balise. Par exemple : <body>
Enfin, on écrit notre contenu après la balise, et ensuite on la referme avec une autre balise du même nom mais avec un slash ( / ) devant le nom : <body> Le contenu placé à l'intérieur de la balise </body>

Exemple de balises paires : 

<body></body> : C'est le "corps" de votre page web.
<div></div> : Permet de créer un élément servant à structurer notre page web et auquel on pourra attribué un id et/ou une ou plusieurs classes.
<h1></h1> : Permet d'écrire un un titre (h1 pour le titre principal, h2 pour un sous-titre et ainsi de suite jusqu'à h6).
<p></p> : Permet d'écrire un paragraphe.
<strong></strong> : Ecrit en gras le texte qui se trouve à l'intérieur de la balise.

Les balises orphelines

Contrairement aux balises fonctionnant par paire, les balises orphelines fonctionnent seules.

Exemple de balises orphelines : 

<br> : Crée un retour à la ligne.
<img> : Permet de placer une image sur notre page web (nécessite des attributs, mais nous verrons ça plus bas).
<hr> : Place un trait qui sépare votre contenu.

Les attributs

Les balises peuvent prendre des attributs. Ces attributs permettent aux balises de savoir quoi faire et de leur passer des informations.
Dans le cas d'une balise image par exemple, nous aurons ceci : 

<img src="emplacement_de_l_image/nom_de_l_image.png" alt="Texte qui définit l'image" >
Dans ce cas ci, l'attribut src (source) permet de passer à la balise img le chemin de l'image qu'elle doit afficher, et l'attribut alt (texte alternatif) permet d'afficher un texte en cas d'erreur lorsque l'image ne s'affiche pas.
Un autre exemple que vous rencontrerez courramment est le suivant : 
<div id="article"></div>
Ici, on donne un id à notre div signifiant qu'il s'agit d'un article. Cette id peut servir à la fois à la naviguation (lien interne à la page) et à la fois au design en le reliant à une feuille de style (CSS).

La différence entre inline et block

Une balise dite "inline" se place parmis du texte, elle ne crée pas de retour à la ligne et, en général, sert à modifier et mettre en forme du texte.
À l'inverse, les balises dites "block" font créer un retour à la ligne. Elles n'ont pas pour but de formater votre texte, mais de structurer votre page.

  • Exemple de balises inline : 
    • <span></span>
    • <strong></strong>
    • <a>
  • Exemple de balises block :
    • <div></div>
    • <h1></h1>
    • <a>
Il existe aussi des balises dites "inline-block" qui se comportent comme des balises inline mais avec des attributs propres aux balises block.
Vous en connaissez déjà une, puisque la balise img en est une !

Le MarkDown et l'HTMLsource

Le MarkDown fonctionne un peu comme de l'HTML mais en version simplifié. En somme, au lieu d'utiliser des balises, on va se servir de caractères particuliers pour mettre en forme notre texte.
Par exemple, pour mettre du texte en gras : 

HTML
 Un élément du texte est <strong>important</strong> !
                    
MarkDown
Un élément du texte est *important* !
Le MarkDown s'écrit dans des documents dont l'extension est en .md (tout comme l'HTML s'écrit dans des documents .html). Là où le MarkDown est lu, il sera converti en HTML.

Quelques exemples supplémentaires :

  • Pour afficher une image : 
    HTML
    <a src="emplacement_de_l_image/nom_de_l_image.png" alt="Texte alternatif">
                            
    MarkDown
    [Texte alternatif](emplacement_de_l_image/nom_de_l_image.png "Description de l'image")
  • Pour écrire une citation : 
     
    HTML
    <blockquote>
    "La soumission n'est pas une option pour ceux qui ont une moustache de champion"
    - Simon Proulx
    </blockquote>
                            
                                    
    MarkDown
    > "La soumission n'est pas une option pour ceux qui ont une moustache de champion" 
> - Simon Proulx
  • Pour afficher une vidéo YouTubesource : 
    HTML
    <iframe width="420" height="315"
src="https://www.youtube.com/embed/id_de_la_video">
</iframe>
                            
    MarkDown
    [![Texte alternatif](https://www.youtube.com/embed/id_de_la_video/0.jpg)](https://www.youtube.com/embed/id_de_la_video)
